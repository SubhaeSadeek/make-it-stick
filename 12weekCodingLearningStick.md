---

# BOOTCAMP GLOBAL RULES (NON-NEGOTIABLE)

## Rule 1 — 70% production minimum

যে কোনো শেখার দিনে input ৩০%, output ৭০%.

## Rule 2 — Every concept must return 5 times

কোনো concept একদিন শিখে ফেলে রাখা যাবে না।

## Rule 3 — Blank screen daily

প্রতিদিন অন্তত ২০ মিনিট no-help coding.

## Rule 4 — Error logging mandatory

প্রতিটি meaningful bug লিখতে হবে.

## Rule 5 — Weekly audit sacred

সপ্তাহিক audit skip করা নিষিদ্ধ.

---

# THE DAILY BOOTCAMP TEMPLATE (MON–SAT)

- 25m concept intake
- 20m blind recall rewrite
- 40m constrained build
- 20m old concept retrieval
- 15m debugging/error harvest
- 15m explain aloud / written reflection

≈ 2 hr 15 min minimum deep block

Optional: +45m free project zone

Sunday = audit + recovery + spaced rebuild.

---

# WEEK 1–2: RETRIEVAL SHOCK & CONSUMPTION DETOX

## Mission: spectator থেকে producer হওয়া

### Objectives

- tutorial dependency কমানো
- দেখে বুঝেছি ভ্রম ভাঙা
- blind recall habit তৈরি

### Daily Special Drill

প্রতিদিন শেখা topic-এর শেষে:

> close everything and rewrite syntax + 2 examples + 1 explanation.

### Blank Screen Challenge

১০–২০ লাইনের tiny utilities zero help লিখুন.

### Week-End Audit

- কোন concept না দেখে লিখতে পারি?
- কোথায় screen blank হয়ে যায়?
- আমি কি এখনও বেশি দেখছি?

---

# WEEK 3–4: MEMORY CONSOLIDATION & SPACED RETURN

## Mission: শেখা জিনিস ফিরিয়ে আনার রাস্তা বানানো

এখন থেকে week 1–2 topics rotate back আসবে.

### New Rule

প্রতিদিন:

- 1 new concept
- 2 old concept recall

### Retrieval Deck Creation

তৈরি করুন:

- syntax cards
- explain-this cards
- missing-line prompts
- bug recognition prompts

### Mini Build Pattern

পুরনো concept + নতুন concept = one micro feature.

### Week-End Audit

১ ঘণ্টা mixed blank recall session.

---

# WEEK 5–6: DELIBERATE STRUGGLE CONDITIONING

## Mission: frustration tolerance build করা

এখন easy flow intentionally ভাঙতে হবে.

### Struggle Rule

প্রতিটি problem-এ minimum 15 মিনিট solo attempt before docs.

### Constraint Drills

- no autocomplete block
- no copy paste
- limited console.log hints only
- rewrite after hint

### Cognitive Goal

panic নয়, analytical persistence.

### Week-End Audit

কোন ধরনের problem-এ আমি খুব তাড়াতাড়ি surrender করি?

---

# WEEK 7–8: PROJECT INTEGRATION WARFARE

## Mission: বিচ্ছিন্ন concept-কে collision zone-এ ফেলা

এখন থেকে সপ্তাহে 2 integration build.

যেখানে অন্তত 3–4 পুরনো concept reuse mandatory.

Examples:

- state + event + storage
- fetch + async + render + error state
- form + validation + condition + DOM update

### Special Drill: Blind Rebuild Day

সপ্তাহে একদিন আগের micro project zero থেকে আবার বানান.

### Week-End Audit

আমি কি feature assemble করতে পারছি, নাকি শুধু isolated snippet জানি?

---

# WEEK 9–10: DEBUGGING INTELLIGENCE & ARCHITECTURE MEMORY

## Mission: code লেখা থেকে code reasoning-এ ওঠা

এখন focus শুধু build নয় — diagnose.

### Mandatory Bug Sheet

প্রতিটি bug:

- symptom
- suspected cause
- actual cause
- permanent lesson

### Reverse Engineering Drill

পুরনো নিজের code খুলে:
লাইন ধরে explain করুন কেন লেখা হয়েছিল.

### Refactor Recall

একই logic cleanerভাবে rewrite.

### Week-End Audit

আমি কি error message দেখে calmer হচ্ছি?

---

# WEEK 11: SPEED + FLUENCY UNDER RETRIEVAL

## Mission: ধীরে বোঝা থেকে দ্রুত cognitive access

এখন timer introduce করুন.

### Timed Drills

- 10 min syntax rewrite
- 15 min feature stub
- 10 min bug locate
- 5 min explain concept

Goal = speed without docs.

### Mixed Recall Marathon

এক সেশনে 8–10 topics random recall.

---

# WEEK 12: CONSOLIDATION TRIAL BY FIRE

## Mission: prove durable developer cognition

এই সপ্তাহে নতুন শেখা কম.

মূল কাজ:

- blank screen rebuilds
- old project reimplementation
- mixed concept challenges
- full self audit

### Final Three Trials

## Trial 1 — Zero Help Coding Test

৯০ মিনিট. docs বন্ধ. একটি useful feature বানান.

## Trial 2 — Explain Like Mentor Test

১০টি concept না দেখে explain করুন.

## Trial 3 — Bug Combat Test

ইচ্ছাকৃত broken code নিয়ে root cause ধরুন.
